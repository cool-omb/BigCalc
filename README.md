# [BIG calculator](https://krkrkrrn.github.io/BIG-calculator/)

## 概要 Overview

[公開ページ](https://krkrkrrn.github.io/BIG-calculator/)

`(1+1)*2` や `2**1000` が、計算できる電卓です。

## 必要要件 Requirement

- Chrome
- Firefox

## 使い方 Usage

ボタン・キーボード入力どちらも可。

## 仕様 Features

- 小数含む計算では、有効数字 10 桁。
- `±2**53` 以上の出力になる計算では、小数点以下の値を計算していません。
